1 से 10 तक का योग

फ़ैक्टोरियल (recursion)
फ़ंक्शन जो खुद को बुलाता है — कार्य / वापस
कार्य फ़ैक्टोरियल(n) { अगर (n <= 1) { वापस 1 } वापस n * फ़ैक्टोरियल(n - 1) } छापो (फ़ैक्टोरियल(5))
output
120
फिबोनाची (fibonacci)
recursion, दूसरे base case के साथ
कार्य फ़िबोनाची(n) { अगर (n <= 1) { वापस n } वापस फ़िबोनाची(n - 1) + फ़िबोनाची(n - 2) } छापो (फ़िबोनाची(7))
output
13
