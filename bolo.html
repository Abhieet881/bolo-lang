बोलो
bolo — literally, "speak"

कोड वैसे लिखो, जैसे तुम बोलते हो।

बोलो एक प्रोग्रामिंग भाषा है जहाँ if, while, function जैसे शब्दों की जगह आप अगर, जब_तक, कार्य जैसे हिंदी शब्दों में कोड लिखते हैं। अंग्रेज़ी जाने बिना भी, अपनी भाषा में सोच कर कोड लिखो।

शुरू करो ↓ शब्दकोश देखो
परिचय

यह किसके लिए है?

ज़्यादातर प्रोग्रामिंग भाषाएँ अंग्रेज़ी keywords पर बनी हैं — if, else, while, function। जो लोग अंग्रेज़ी में सहज नहीं हैं, उनके लिए यह एक अतिरिक्त रुकावट है, असली प्रोग्रामिंग सीखने से पहले ही। बोलो उस रुकावट को हटाने की कोशिश है।

हिंदी भाषियों के लिए

स्कूल के बच्चे और शुरुआती सीखने वाले, जिनके लिए हिंदी पहली भाषा है — वे logic और syntax पर ध्यान दे सकते हैं, अंग्रेज़ी अनुवाद पर नहीं।

developers के लिए

यह Go में लिखा गया एक पूरा tree-walking interpreter है — lexer, parser, evaluator, closures, recursion सब काम करता है। एक भाषा कैसे बनती है, यह समझने का एक असली project।

शुरुआत

बोलो चलाना शुरू करो

चार आसान चरण — पूरी तरह मुफ़्त, कोई भुगतान नहीं।

Go install करो
go.dev/dl से Go download करके install करो। फिर terminal में जाँचो:
go version
बोलो project लो
repository clone करो या ZIP download करके अपने folder में रखो।
build करो
project folder में terminal खोलो और चलाओ:
go build -o bolo.exe main.go
अपना पहला प्रोग्राम चलाओ
hello.bolo नाम की फ़ाइल बनाओ, यह लिखो, फिर चलाओ:
मान क = 10
मान ख = 5
छापो (क + ख)
./bolo.exe hello.bolo
उदाहरण

असली चलते हुए प्रोग्राम

यह सब हाथ से लिखे और चलाए गए — नीचे असली output है।

सम या विषम (even / odd)
शर्त जाँचना — अगर / नहीं_तो
मान संख्या = 7 अगर (संख्या % 2 == 0) { छापो ("सम संख्या") } नहीं_तो { छापो ("विषम संख्या") }
output
विषम संख्या
1 से 10 तक का योग
लूप और चर का दोबारा उपयोग — जब_तक
मान योग = 0 मान i = 1 जब_तक (i <= 10) { योग = योग + i i = i + 1 } छापो ("योग है:") छापो (योग)
output
योग है: 55
फ़ैक्टोरियल (recursion)
फ़ंक्शन जो खुद को बुलाता है — कार्य / वापस
कार्य फ़ैक्टोरियल(n) { अगर (n <= 1) { वापस 1 } वापस n * फ़ैक्टोरियल(n - 1) } छापो (फ़ैक्टोरियल(5))
output
120
