बोलो चलाना शुरू करो

पाँच चरण, पूरी तरह मुफ़्त, कोई भुगतान नहीं। Windows, Mac, और Linux तीनों पर काम करता है।

Go इंस्टॉल करो
go.dev/dl से अपने operating system के लिए Go installer download करो और चलाओ। फिर एक नया terminal खोलकर जाँचो:
go version
एक code editor लो
VS Code मुफ़्त है (code.visualstudio.com)। चाहो तो Antigravity CLI जैसा AI coding agent भी जोड़ सकते हो, उसका free tier भी काम करता है।
बोलो project लो
repository clone करो या ZIP download करके किसी folder में रखो, जैसे:
git clone <repo-url> bolo
cd bolo
build करो
project folder के अंदर terminal में चलाओ:
go mod tidy
go build -o bolo.exe main.go
Mac/Linux पर फ़ाइल का नाम सिर्फ़ bolo होगा, .exe नहीं।
अपना पहला प्रोग्राम चलाओ
examples/hello.bolo नाम की फ़ाइल बनाओ:
मान क = 10
मान ख = 5
छापो (क + ख)
फिर चलाओ:
./bolo.exe examples/hello.bolo
समस्या निवारण

अक्सर आने वाली दिक़्क़तें

"The system cannot find the file specified"
इसका मतलब है फ़ाइल का path ग़लत है। जाँचो कि फ़ाइल कहाँ है — terminal में pwd चलाकर अपना current folder देखो, फिर सही path से फ़ाइल चलाओ।
'go' is not recognized / go: not found
Go सही से install नहीं हुआ, या terminal को restart करना बाक़ी है। Go install करने के बाद हमेशा एक नया terminal खोलो।
कई छापो calls एक ही लाइन में जुड़ जाते हैं
यह एक पुराना bug था जो ठीक कर दिया गया है। अगर फिर भी दिखे, तो latest project version download करो।
