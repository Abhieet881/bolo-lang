शुरुआत से आगे तक

छह पाठ, आसान से कठिन तक। हर पाठ ख़ुद टाइप करके चलाओ — देखकर नहीं, हाथ से लिखकर सीखो।

पाठ १ · शुरुआती

चर और गणित

मान से एक चर बनाओ, उसमें संख्या रखो, और छापो से परिणाम दिखाओ।

मान क = 10
मान ख = 5
छापो (क + ख)
पाठ २ · शुरुआती

शर्त (if / else)

अगर किसी शर्त को जाँचता है; अगर वह सच नहीं है, तो नहीं_तो वाला हिस्सा चलता है।

मान संख्या = 7
अगर (संख्या % 2 == 0) {
    छापो ("सम संख्या")
} नहीं_तो {
    छापो ("विषम संख्या")
}
पाठ ३ · मध्यम

लूप (जब_तक)

जब_तक एक शर्त के सच होने तक बार-बार चलता है। यहाँ 1 से 10 तक जोड़ा जा रहा है।

मान योग = 0
मान i = 1
जब_तक (i <= 10) {
    योग = योग + i
    i = i + 1
}
छापो (योग)
पाठ ४ · मध्यम

नेस्टेड शर्तें (FizzBuzz)

एक लूप के अंदर कई अगर/नहीं_तो होने पर भी कोड साफ़ रहता है।

मान i = 1
जब_तक (i <= 15) {
    अगर (i % 15 == 0) {
        छापो ("FizzBuzz")
    } नहीं_तो अगर (i % 3 == 0) {
        छापो ("Fizz")
    } नहीं_तो अगर (i % 5 == 0) {
        छापो ("Buzz")
    } नहीं_तो {
        छापो (i)
    }
    i = i + 1
}
पाठ ५ · उन्नत

फ़ंक्शन और recursion

कार्य से एक फ़ंक्शन बनता है, जो वापस से मान लौटाता है — और ख़ुद को भी बुला सकता है।

कार्य फ़ैक्टोरियल(n) {
    अगर (n <= 1) {
        वापस 1
    }
    वापस n * फ़ैक्टोरियल(n - 1)
}
छापो (फ़ैक्टोरियल(5))
पाठ ६ · उन्नत

array के साथ काम करना

लंबाई और के_लिए को साथ में इस्तेमाल करके किसी सूची के हर तत्व पर काम करो।

मान फल = ["आम", "केला", "सेब"]
मान i = 0
जब_तक (i < लंबाई(फल)) {
    छापो (फल[i])
    i = i + 1
}
आगे क्या
